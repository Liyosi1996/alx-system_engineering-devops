This is a readme file for 0x03-shell_variables_expansions
0. 0-alias file Creates a script that creates an alias.
1. 1-hello_you file Creates a script that prints hello user, where user is the current Linux user.
2. 2-path files adds /action to the PATH. /action should be the last directory the shell looks into when looking for a program.
3. 3-paths file creates  a script that counts the number of directories in the PATH.
4. 4-global_variables file creates a script that lists environment variables
5. 5-local_variables file creates a script that lists all local variables and environment variables, and functions.
6. 6-create_local_variable file creates a script that creates a new local variable.
7. 7-create_global_variable file creates a script that creates a new global variable.
8. 8-true_knowledge file writes a script that prints the result of the addition of 128 with the value stored in the environment variable TRUEKNOWLEDGE, followed by a new line.
9. 9-divide_and_rule file writes a script that prints the result of POWER divided by DIVIDE, followed by a new line.
10. 10-love_exponent_breath file writes a script that displays the result of BREATH to the power LOVE
11. 11-binary_to_decimal file writes a script that converts a number from base 2 to base 10.
12. 12-combinations file creates a script that prints all possible combinations of two letters, except oo.
13. 13-print_float file writes a script that prints a number with two decimal places, followed by a new line.
14. 100-decimal_to_hexadecimal file writes a script that converts a number from base 10 to base 16.
15. 101-rot13 file writes a script that encodes and decodes text using the rot13 encryption. Assume ASCII.
16. 102-odd file writes a script that prints every other line from the input, starting with the first line.
17. 103-water_and_stir file writes a shell script that adds the two numbers stored in the environment variables WATER and STIR and prints the result.
